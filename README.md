# Soul Echoes: The All-In-One Healing & Accessibility Sanctuary 🌿✨

Soul Echoes is a comprehensive digital sanctuary and emotional wellness application built to be fully inclusive for all communication levels, languages, and accessibility profiles—including deaf, blind, nonverbal, dyslexic, and neurodivergent users.

## 🌟 Core App Ecosystem & Architecture

### 🧠 1. Brain Dump (AI Advocate Companion)
*   **Trauma-Informed Guidance:** Powered by a built-in Lovable AI Edge function gateway operating with a "Soul Echo" persona that gently processes user thoughts, handles crisis flags, and dynamically routes users to specific healing spaces.
*   **Express through Colors & Symbols:** A specialized visual input canvas. Users tap or drag shapes and color palettes to express nonverbal emotions. The AI companion reads visual configurations (e.g., mapping dark blue to melancholy) and responds warmly.

### 🗣️ 2. Comprehensive Input Method Bar
Provides absolute multi-modal communication flexibility across five core methods:
*   `🗣️ Speak It` – Voice input supported by an optimized, async-loaded browser Text-to-Speech (TTS) engine that matches vocal locales to native languages.
*   `🤟 Sign It` – Built-in video configuration/toggle mechanics for sign-language communication.
*   `👆 Point It` – A massive, categorized grid of emotion and necessity cards (Feelings like Sad/Angry/Anxious, Body Needs, and Quick Assistance indicators). Tapping any card directly sends an actionable update to the AI.
*   `⌨️ Type It` – Highly accessible, large touch-target text interface for standard keyboard interaction.
*   `🔌 Connect Device` – Fully integrated Bluetooth/USB interface to hook up external AAC hardware, eye-gaze tracking modules, or alternative accessibility equipment.

### 🌿 3. The 9 Specialized Healing Rooms
1.  **Journal** – Fully private personal reflection and emotional processing vault.
2.  **Breathe** – Composed of 8 scrollable section cards leading to interactive layout pages detailing meditation basics (Breath Focus, Chakras, Body Scan, Affirmations, Sound Baths), deep logs for all 12 Chakras plus Earth Star (Chakra 0), vagus nerve stimulation, and Solfeggio frequency therapies.
3.  **Unspoken Chamber** – A dedicated release zone for heavy feelings left unsaid.
4.  **Shadow Work** – Introspective self-exploration tracks and emotional prompt structures.
5.  **Wisdom** – Curated spiritual guidance archives, universal quotes, and philosophical tracks.
6.  **Spiritual Tools** – Modern energetic alignment mechanics and modules.
7.  **Community** – Space to safely connect with people sharing similar dreams.
8.  **Practitioner Connect** – Interactive network portal bridging users to verified wellness professionals.
9.  **Crisis Counselor** – Direct interface linking users to immediate, certified support nets.

### ♿ 4. Advanced Dyslexia & Sensory Comfort Configuration
*   **Global Typography Selector:** Accessible instantly from settings to override the entire application layout into `OpenDyslexic` (weighted bottoms), `Comic Sans / Verdana` (high-legibility spacing), or `System Default` fonts.
*   **Calm Tones Theme:** A universal setting toggle that instantly converts hard dark-mode interfaces into soft cream background tones with charcoal text for high-comfort, low-sensory strain reading.
*   **Asynchronous TTS System:** Deeply optimized Web Speech API layer utilizing `voiceschanged` event monitoring to eliminate silent voice failures on mobile and desktop web environments.

## 🔐 5. Database, Security, & Payment Pipelines
*   **Supabase Security Architecture:** Hardened relational database layout implementing row-level security (RLS) constraints on the `distress_signals` table. Features localized CHECK constraints (`situation_code` ranges 111–555) and an explicit `user_roles` system (admin, dispatcher, reviewer) to protect user identity.
*   **Dual-Track Monetization Hub:** Built-in pricing system layout cleanly split into two distinct models:
    *   *Track 1 (Individual Users):* 6 tiers starting from a Free 33-day trial tier up to a $9/mo Ultimate Beta package.
    *   *Track 2 (Professionals & Healers):* 5 income-contingent professional tiers starting at $2/mo designed for verified holistic health specialists.
