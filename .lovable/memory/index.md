# Project Memory

## Core
Soul Echoes — spiritual healing & emotional wellness app. Accessibility-first (deaf, blind, nonverbal, neurodivergent).
Dark sanctuary theme: bg #151a23, primary amber hsl(30,70%,50%), sage green, indigo accent. Nunito display + Atkinson Hyperlegible body.
Lovable Cloud + Lovable AI (gemini-3-flash-preview). Brain Dump = main AI chat page.
9 healing rooms: Journal, Breathe, Unspoken Chamber, Shadow Work, Wisdom, Spiritual Tools, Community, Practitioner Connect, Crisis Counselor.
Communication methods stored as array (up to 3). Brain Dump shows tabs per selected method.
Distress signal button always visible in bottom-right corner.
Brain Dump + Crisis Counselor always free. Other rooms show lock icon.
