# Soul Echoes Icon Bible Journal

## Purpose

This document tracks the creation, meaning, evolution, and usage rules for all Soul Echoes visual icons.

The goal is to keep every icon connected to the same emotional language, symbolism, and brand identity.

Soul Echoes is built around themes of:

* Healing
* Memory
* Connection
* Reflection
* Growth
* Human stories
* Inner voice
* Emotional resilience

---

# Icon Philosophy

Every Soul Echoes icon should feel:

* Timeless
* Meaningful
* Calm but powerful
* Human-centered
* Connected to healing and transformation

Icons should not feel like generic app graphics. Each symbol should represent an emotional concept or journey.

---

# Core Visual Language

## Shape Language

Preferred:

* Soft curves
* Organic forms
* Circular movement
* Flowing lines
* Balanced symmetry

Avoid:

* Harsh angles
* Overly technical shapes
* Corporate-looking symbols

---

# Icon Categories

## 1. Memory Icons

Purpose:
Represent stories, experiences, and moments preserved through Soul Echoes.

Symbol ideas:

* Echo waves
* Rings
* Ripples
* Open journals
* Light trails

Meaning:
Every memory creates an echo.

---

## 2. Healing Icons

Purpose:
Represent emotional growth and restoration.

Symbol ideas:

* Rising light
* Open hands
* Leaves
* Hearts integrated with natural elements
* Transformation symbols

Meaning:
Healing is a journey, not a destination.

---

## 3. Connection Icons

Purpose:
Represent relationships between people and shared experiences.

Symbol ideas:

* Constellations
* Linked circles
* Threads
* Bridges
* Stars

Meaning:
Every person is connected through shared echoes.

---

## 4. Journey Icons

Purpose:
Represent progress and personal transformation.

Symbol ideas:

* Paths
* Mountains
* Spirals
* Rising shapes
* Horizons

Meaning:
Growth happens one step at a time.

---

# Icon Rules

Every new icon should answer:

1. What emotion does this represent?
2. What story does this symbol tell?
3. Does it belong inside the Soul Echoes universe?
4. Would someone understand its feeling without explanation?

---

# Design Notes

Icons should work in:

* App interface
* Mobile navigation
* Social media
* Marketing materials
* Merchandise
* Future Rise Up Healing ecosystem

Icons should remain recognizable when:

* Large
* Small
* Single color
* Used as an app button
