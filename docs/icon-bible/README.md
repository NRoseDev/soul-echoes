# Soul Echoes Icon Bible

## Purpose

This document defines the visual language for all icons used in Soul Echoes.

All icons across the app must follow this system to ensure consistency, quality, and emotional alignment.

This is the single source of truth for icon design.

---

## Philosophy

Soul Echoes is a healing sanctuary.

Icons are not generic UI elements.

They are visual expressions of:
- peace
- curiosity
- healing
- awareness
- inner discovery

Every icon should feel intentional, calm, and meaningful.

---

## Core Principles

- Icons must feel like they belong to the same family
- Preserve symbolism from reference images
- Do not convert meaningful visuals into generic clip art
- Simplify without losing essence
- Maintain emotional impact

---

## Visual Style

- Premium
- Elegant
- Spiritual
- Minimal but expressive
- Luxury wellness aesthetic

---

## Line Weight

- All icons use consistent stroke thickness
- No icon should appear heavier or lighter than another

---

## Glow

- One chakra color per icon
- Soft glow only
- No harsh effects
- No rainbow gradients

---

## Geometry

- Smooth curves
- Rounded edges
- Balanced proportions
- Even spacing

---

## Simplification Rule

When using a reference image:

DO:
- Preserve concept
- Preserve symbolism
- Preserve composition

DO NOT:
- Replace with generic icons
- Over-simplify into clip art
- Remove meaningful details

---

## Visual Balance

Target style:

Elegant Luxury Minimalism

Not too detailed.
Not too simple.

---

## Icon Framing

- Consistent padding
- Centered composition
- Equal visual weight across all icons

---

## Accessibility

Icons must remain recognizable at:
- 32px
- 48px
- 64px

Icons should not rely only on color to be understood.

---

## System Rule

Do not create or modify icons without following this document.
