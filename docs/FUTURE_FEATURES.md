# 🌙 Dream Journal

**Status:** Planned

## Purpose

The Dream Journal is a dedicated healing space where users can safely record their dreams without interruption. After a dream is saved, Soul Echo quietly prepares personalized reflections for the user to explore whenever they feel ready.

The user is never interrupted while recording a dream.

The user decides when, or if, they want to view Soul Echo's reflections.

## Planned Features

- Private dream entries
- Voice and text dream recording
- Dream title (optional)
- Date and time tracking
- AI-generated dream summary
- Symbolic reflections
- Emotional theme recognition
- Personalized reflection prompts
- Recurring dream pattern tracking
- Personalized symbol library based on previous dreams
- Suggested healing rooms when appropriate
- Optional connection to a verified practitioner for deeper dream exploration

## Core Principles

- No AI interruptions while writing.
- Soul Echo prepares reflections quietly in the background.
- Healing unfolds one layer at a time.
- The user remains in complete control of when to continue.
- Reflections are offered as possibilities for self-exploration, never presented as absolute truth.
